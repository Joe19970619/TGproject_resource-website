## 项目启动

``` bash

在本地shell或git bash里分别运行如下命令：

1.git clone git@github.com:TGclub/TGproject_resource-website.git

2.cd TGproject_resource-website

3.cnpm install(或者npm install) （本地没有node的先去下载node

4.npm run dev

5.在浏览器输入 http://localhost:8080
```
